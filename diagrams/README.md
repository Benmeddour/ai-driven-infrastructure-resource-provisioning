### Templates

The following files are **reusable templates** that can be helpful for creating new diagrams or extending the project:

- **`template_agent_pipline.drawio`** - Template for agent pipeline workflows  
  A reusable template for designing AI agent pipelines and CrewAI-based workflows, showing the flow between user interaction, agent processing, and tool execution.

- **`data_center_template.drawio`** - Template for data center infrastructure  
  A template for designing data center architectures with redundancy, scalability, and security components.

- **`data_center_flow_diagram.drawio`** - Data center workflow example  
  An example diagram showing data flow in a data center environment with high availability, load balancing, and disaster recovery considerations.
